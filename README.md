This is a plug‑and‑play **GitHub Resume + Portfolio** built for **GitHub Pages**.

## Deploy (GitHub Pages)
1. Create a new repo named **`YOUR-GITHUB.github.io`** (recommended).
2. Upload the contents of this template.
3. Go to **Settings → Pages**
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/docs**
4. Your site will be live at `https://YOUR-GITHUB.github.io/`.

## Update your info
- Edit `docs/data/profile.json`
- Edit `docs/data/projects.json`

## Optional
- Put a resume PDF at `docs/assets/resume.pdf` and enable the “Resume PDF” button in `docs/index.html`.
